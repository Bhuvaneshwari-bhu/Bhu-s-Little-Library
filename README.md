# 📚 BHU Books — Illustrated Children's Book Showcase

A single-page portfolio for showcasing illustrated children's books and coloring books in a storybook-inspired web experience.

This project transforms a simple catalog into an interactive bookshelf where each book opens into a detailed view with smooth animations and an Amazon link.

Built using modern frontend tools with a focus on UI/UX, animation, and visual storytelling.

## About the Project

This project is designed to:

* Showcase illustrated storybooks
* Present coloring books and children's stories
* Experiment with animation-driven UI
* Create a storybook-like browsing experience
* Turn static content into an interactive visual journey

The goal is to make browsing books feel like opening a magical library rather than using a standard website.

---

## Features

* Interactive animated bookshelf UI
* Smooth Framer Motion transitions
* Click-to-open book detail view
* Layout animations between shelf and detail view
* Responsive design for all devices
* Clean pastel storybook-inspired UI
* Fast Vite build performance

## Purpose of This Project

This project is not just a bookstore. It is:

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

The project runs with placeholder data from:

```
src/data/fallbackBooks.js
```

---

## Add Your Own Books (No Backend Required)

Edit the file:

```
src/data/fallbackBooks.js
```

Update:

* title
* subtitle
* description
* cover image
* Amazon link

Then redeploy to Vercel.
